Arrays !

  • JavaScript arrays are fundamental data structures that allow you to store and manipulate collections of data. Is used to store multiple values in a single variable.
  • Arrays are mutable, meaning you can change their elements.
  • It is a common practice to declare arrays with the const keyword;
  • Arrays in JavaScript are a special type of object. Although you can use arrays like objects with properties, arrays are optimized for ordered data, whereas objects are better for key-value pairs.
  • The typeof an array is "object", which can be surprising. To check if a variable is an array, use Array.isArray(variable).
  • Adding or removing elements from the beginning of an array using shift() or unshift() is generally slower than pop() or push() because it requires reindexing all the elements.
  • Many array methods mutate the original array (e.g., push, pop, splice), while others return new arrays (e.g., map, filter, slice).
  • Higher-order functions like map(), filter(), and reduce() are powerful tools for array manipulation and can lead to cleaner and more functional code.
  • Arrays can contain objects, functions, other arrays, and mixed types.
    let complexArray = [1, {name: 'John'}, [2, 3], () => 'hello'];


  • Most important methods:

    • MAP:

      * Return a new array.

      * Doesn't change the original array.

      * Calling a function for every array element.

      const ages = [12, 7, 54, 83, 11, 48];
      const doubleAge = ages.map((age) => age * 2);
      console.log(doubleAge); // [24, 14, 108, 166, 22, 96]
    • FILTER:

      * Return a new array.

      * Doesn't change the original array.

      * Filled with elements that pass a test provided by a function.

      const ages = [12, 7, 54, 83, 11, 48];
      const canDrink = ages.filter((age) => age > 21);
      console.log(canDrink); // [54, 83, 48]
    • REDUCE:

      * Returns a single value: the function's accumulated result.

      * Doesn't change the original array.

      const ages = [12, 7, 54, 83, 11, 48];
      const sumAges = ages.reduce((acc, value) => acc + value, 0);
      console.log(sumAges); // 215
    • SORT:

      * Overwrites the original array.

      * By default, sort() sorts elements as strings; This can lead to unexpected results when sorting numbers.

      * To correctly sort numbers, you should provide a comparison function.

      const arr = [10, 5, 20];
      arr.sort((a, b) => a - b); console.log(arr); // [5, 10, 20]
    • FLAT:

      * Return a new array.

      * Doesn't change the original array.

      * With the sub-array elements concatenated into it.

      const arr = [1, [2, [3, 4]]];
      const flatArr = arr.flat(2);
      console.log(flatArr); // [1, 2, 3, 4]
    • REVERSE:

      * Overwrites the original array.

      const ages = [12, 7, 54, 83, 11, 48];
      console.log(ages.reverse()); // [48, 11, 83, 54, 7, 12]
    • PUSH:

      * Overwrites the original array.

      * Adds new items to the end of an array.

      * Changes the length of the array.

      * Returns the new length.

      const vegetables = ["tomato", "potato"];
      const newVeg = vegetables.push("pumpkin");
      console.log(vegetables); // ["tomato", "potato", "pumpkin"]
      console.log(newVeg); // 3
    • POP:

      * Overwrites the original array.

      * Removes (pops) the last element of an array.

      * Returns the removed element.

      const vegetables = ["tomato", "potato"];
      const newVeg = vegetables.pop();
      console.log(vegetables); // ["tomato"]
      console.log(newVeg); // "potato"
    • UNSHIFT:

      * Overwrites the original array.

      * Adds new elements to the beginning of an array.

      * Returns the new length.

      const vegetables = ["tomato", "potato"];
      const newVeg = vegetables.unshift('pumpkin');
      console.log(vegetables); // ["pumpkin", "tomato", "potato"]
      console.log(newVeg); // 3
    • SHIFT:

      * Overwrites the original array.

      * Removes the first item of an array.

      * Returns the shifted element.

      const vegetables = ["tomato", "potato"];
      const newVeg = vegetables.shift();
      console.log(vegetables); // ["potato"]
      console.log(newVeg); // "tomato"
